# Navigation & Mobile Gestures

Prismedia's navigation is yours to arrange, and your arrangement is saved on the **server** — so it follows you across every device and browser, not just the one you set it on.

## The desktop sidebar

The sidebar groups destinations into sections. Out of the box:

| Section | Destinations |
| --- | --- |
| **Overview** | Dashboard, Search |
| **Video** | Movies, Series, Videos |
| **Images** | Galleries, Images |
| **Audio** | Artists, Audio |
| **Books** | Books, Comics, eBooks |
| **Browse** | People, Studios, Tags, Collections |
| **Operate** | Files, Identify, Plugins, Jobs, Settings |

### Editing the layout

Click the **pencil** at the bottom of the rail to enter edit mode, where you can:

- **Reorder** sections and the items within them (drag and drop).
- **Rename** sections and **create** your own.
- **Move** items between sections.
- **Hide** destinations you never use (they reappear in edit mode so you can switch them back on).
- **Reset** to the default layout.

### Collapsing sections

Click a section header to fold its items away. The collapsed/expanded state is saved alongside the rest of your layout, so the sidebar reopens exactly how you left it on every device.

### Footer

The rail footer shows the running **version and channel** (and lights up when an update is available, linking to the changelog) plus a link to this documentation.

## Mobile navigation

On phones, navigation lives in a **bottom bar** that clears modern phone notches, plus a **drawer** with everything else.

### The bottom bar

Pin up to **four** favorite destinations to the bottom bar. A **More** button opens the full drawer.

### Gestures

| Gesture | Action |
| --- | --- |
| **Swipe up** on the bottom bar (or tap the animated chevron) | Open the navigation drawer. |
| **Pull down** on the drawer handle | Dismiss the drawer. |
| In the drawer: **star** a destination | Pin it to the bottom bar (max 4); reorder with a live preview. |
| In the drawer | Hide, reorder, and regroup destinations, just like desktop edit mode. |

Reader and lightbox gestures (tap zones, swipe to turn/dismiss) are covered in [Playback & Reading](./playback.md). Thumbnail scrubbing on touch: a tap opens the item, a vertical drag scrolls, and a horizontal drag scrubs the preview.

## Command palette

Press `Cmd+K` (macOS) or `Ctrl+K` (elsewhere) from any page to jump across the whole library. Results are grouped by category with a "See all" link to the full search.
